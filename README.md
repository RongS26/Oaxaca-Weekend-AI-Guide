# Oaxaca Weekend AI Guide

[English](README.md) | [中文](README.zh-CN.md)

A small AI-assisted travel guide project for a weekend in Oaxaca.

This project documents how a personal travel idea became a bilingual guide and a practical weekend playbook through iterative AI collaboration.

## What This Is

This is not only a travel itinerary.

It is a lightweight case study of using AI to turn:

```text
travel intent
-> cultural context
-> bilingual guide
-> preference-aware refinement
-> weekend playbook
-> reusable personal workflow
```

## Why I Kept It

The durable part is not "where to go in Oaxaca."

The durable part is the interaction pattern:

- Use AI to move from generic travel recommendations to place-specific cultural understanding.
- Ask for a usable artifact with detailed schedule, ticket links, and practical constraints.
- Refine the result with personal context, such as what is redundant given life in Mexico City and what is not worth a paid tour.
- Iterate from a strong first guide into a more personalized weekend playbook.
- Preserve final PDFs and process notes so future trips start from a better template.

## Reusable Idea

AI is most useful here when it is treated as a guide-making collaborator:

1. Ask for cultural and neighborhood logic first.
2. Ask for a concrete artifact with schedule details and links for ticketed items.
3. Challenge recommendations using personal context and local comparisons.
4. Iterate into a playbook with timing, backups, and decision points.
5. Save both the artifact and the workflow.

## Notes

This project is intentionally small. It is a personal AI practice artifact, not a complete travel product.
